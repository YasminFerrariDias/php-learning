# PHP Exercises

Exercícios desenvolvidos no ensino médio durante os primeiros contatos com desenvolvimento web back-end em PHP.

## Requisitos

- PHP 7.4 ou superior
- Servidor local: [XAMPP](https://www.apachefriends.org/) ou [WAMP](https://www.wampserver.com/)

## Como Executar

1. Copie os arquivos para o diretório `htdocs` (XAMPP) ou `www` (WAMP)
2. Inicie o serviço Apache
3. Acesse via navegador: `http://localhost/nome-do-arquivo.php`

## Temas Abordados

- Formulários HTML com método POST
- Validação e sanitização de entradas
- Estruturas condicionais e cálculos
- Simulações financeiras
- Sessões PHP
- Efeitos visuais com CSS hover
- Sistema de autenticação de usuários com cadastro, login e busca
